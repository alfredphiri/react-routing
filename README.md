# React Streaming App

A modern, high-performance streaming application interface built with React and Vite. This project features a premium user interface with smooth animations, responsive design, and a seamless browsing experience for movies and series.

## üöÄ Features

- **Immersive UI**: Dark-themed, premium aesthetic inspired by top streaming platforms.
- **Smooth Animations**: Powered by `framer-motion` for fluid page transitions and micro-interactions.
- **Content Browsing**: Dedicated pages for Movies, Series, and Latest releases.
- **Interactive Components**: Horizontal scrolling rows, hover effects, and dynamic content presentation.
- **Authentication**: Sign In and Sign Up pages with form validation and visual polish.
- **Responsive Design**: Fully optimized for various screen sizes using Tailwind CSS.
- **Routing**: Client-side routing with `react-router-dom` for a single-page application experience.
- **Theming**: Built-in theme context for consistent styling management.

## üì¶ Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/alfredphiri/react-routing.git
    cd react-routing
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
